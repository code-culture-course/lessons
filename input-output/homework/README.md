# Домашнее задание: Приложение для заметок

## Обзор

Создайте полное приложение для заметок с интерфейсом командной строки, которое позволяет пользователям создавать, просматривать и управлять текстовыми заметками. Это домашнее задание объединяет все концепции из урока: консольный ввод/вывод, операции с файлами, JSON и обработку ошибок.

## Цели обучения

Выполнив это домашнее задание, вы сможете:
- Создать реальное, полезное CLI приложение
- Практиковать дизайн программ, управляемых меню
- Использовать JSON для сохранения данных
- Корректно обрабатывать ошибки пользователя
- Структурировать код с помощью функций

## Требования

### Основные функции

Ваше приложение для заметок должно поддерживать:

1. **Просмотр всех заметок**
   - Отображать нумерованный список всех заголовков заметок
   - Показывать "Заметок пока нет" если список пуст

2. **Создать новую заметку**
   - Запросить заголовок заметки
   - Запросить содержимое заметки (может быть несколько строк)
   - Сохранить с отметкой времени (можно использовать простой счётчик или пропустить для упрощения)

3. **Просмотр деталей заметки**
   - Запросить какую заметку просмотреть (по номеру)
   - Отобразить полную заметку (заголовок и содержимое)

4. **Удалить заметку**
   - Запросить какую заметку удалить (по номеру)
   - Подтвердить удаление
   - Обновить сохранённые данные

5. **Выход**
   - Сохранить все данные перед выходом
   - Отобразить прощальное сообщение

### Сохранение данных

- Хранить заметки в JSON файле (`notes.json`)
- Загружать заметки при запуске программы
- Сохранять заметки после каждого изменения (создание/удаление)

### Структура данных

Каждая заметка должна быть словарём с:
```python
{
    "title": "Заголовок заметки",
    "content": "Содержимое заметки здесь..."
}
```

Хранить все заметки в списке:
```python
[
    {"title": "Список покупок", "content": "Молоко\nЯйца\nХлеб"},
    {"title": "Идеи", "content": "Создать игру\nИзучить Python\nПрочитать книгу"}
]
```

### Обработка ошибок

Ваша программа должна обрабатывать:
- Некорректные выборы в меню
- Некорректные номера заметок
- Отсутствующий или повреждённый JSON файл
- Пустые заголовки или содержимое заметок
