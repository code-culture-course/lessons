# Мини-проект: Менеджер списка задач

Этот мини-проект демонстрирует как построить полное CLI приложение шаг за шагом.

## Что мы создаём

Простой менеджер списка задач который:
- Отображает меню
- Позволяет пользователям добавлять и просматривать задачи
- Сохраняет задачи в файл (JSON)
- Загружает задачи при запуске программы
- Корректно обрабатывает ошибки

## Путь обучения

Каждый файл основывается на предыдущем:

1. **step1_menu.py** — Базовая структура меню
2. **step2_tasks_in_memory.py** — Хранение задач в списке (не сохраняется)
3. **step3_save_to_file.py** — Сохранение задач в текстовый файл
4. **step4_json_persistence.py** — Использование JSON для лучшей структуры
5. **step5_error_handling.py** — Обработка некорректного ввода и ошибок файлов

Запустите каждый файл чтобы увидеть как программа эволюционирует!

## Запуск проекта

```bash
# Начните с шага 1
python step1_menu.py

# Прогрессируйте через каждый шаг
python step2_tasks_in_memory.py
python step3_save_to_file.py
python step4_json_persistence.py
python step5_error_handling.py
```

## Ключевые выводы

К концу вы поймёте:
- Как структурировать программу управляемую меню
- Как сохранять данные между запусками
- Почему JSON лучше простого текста для структурированных данных
- Как обрабатывать ошибки без сбоев
