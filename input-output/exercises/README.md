# Упражнения по вводу/выводу

Практикуйте свои навыки ввода/вывода с этими упражнениями. Каждое упражнение основывается на концепциях из урока.

## Инструкции

1. Выполняйте упражнения по порядку
2. Запускайте свой код для тестирования
3. Сравнивайте ваше решение с концепциями из основного урока
4. В каждом файле есть docstring, объясняющий что нужно реализовать

## Список упражнений

### Консольный ввод/вывод
- **Упражнение 1**: [exercise_01_greeting.py](exercise_01_greeting.py) - Читать имя и возраст, отобразить приветствие
- **Упражнение 2**: [exercise_02_calculator.py](exercise_02_calculator.py) - Простой калькулятор с валидацией ввода
- **Упражнение 3**: [exercise_03_menu_loop.py](exercise_03_menu_loop.py) - Построить интерактивное меню

### Операции с файлами
- **Упражнение 4**: [exercise_04_write_log.py](exercise_04_write_log.py) - Записать записи в лог-файл
- **Упражнение 5**: [exercise_05_read_log.py](exercise_05_read_log.py) - Читать и отображать записи лога
- **Упражнение 6**: [exercise_06_line_counter.py](exercise_06_line_counter.py) - Посчитать строки в файле

### JSON
- **Упражнение 7**: [exercise_07_save_profile.py](exercise_07_save_profile.py) - Сохранить профиль пользователя как JSON
- **Упражнение 8**: [exercise_08_load_profile.py](exercise_08_load_profile.py) - Загрузить и отобразить JSON профиль

### Обработка ошибок
- **Упражнение 9**: [exercise_09_safe_input.py](exercise_09_safe_input.py) - Обработать некорректный числовой ввод
- **Упражнение 10**: [exercise_10_safe_file.py](exercise_10_safe_file.py) - Обработать отсутствующие файлы корректно

## Запуск упражнений

```bash
# Запустить каждое упражнение
python exercise_01_greeting.py
python exercise_02_calculator.py
# ... и так далее
```

## Советы

- Тестируйте свой код с разными входными данными
- Попробуйте сломать вашу программу (некорректный ввод, отсутствующие файлы)
- Добавляйте обработку ошибок где уместно
- Держите ваш код чистым и читаемым
