# OOP Exercises

Complete these exercises to practice Object-Oriented Programming concepts.

## How to Work on Exercises

1. Read each exercise description carefully
2. Create the solution file in this `exercises/` directory
3. Run your code to test it
4. Compare your solution with the expected output
5. Move on to the next exercise

## Tips

- Start simple and build up gradually
- Test your code frequently
- Don't worry about making it perfect‚Äîfocus on getting it working first
- If you get stuck, review the examples in the `examples/` directory
- Ask for help if needed!
